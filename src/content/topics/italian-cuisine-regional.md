意大利没有"意大利菜"—— 只有地区菜。1861 年统一前意大利被分成 9 个独立政体，每个地方独立发展饮食 1000 多年。今天的"意大利菜"是 20 个大区 + 100 多个省的拼贴。

5 个主要地区饮食的关键差异：
- **罗马 / 拉齐奥**：羊肉 + 内脏 + 朝鲜蓟（cacio e pepe / carbonara / trippa）
- **托斯卡纳 / 佛罗伦萨**：牛肉 + 豆 + 橄榄油（bistecca / ribollita / pici）
- **威尼斯 / 威尼托**：海鲜 + 香料贸易遗产（sarde in saor / risi e bisi / cicchetti）
- **米兰 / 伦巴第**：黄油 + 米饭 + 肉（risotto alla milanese / cotoletta / ossobuco）
- **那不勒斯 / 坎帕尼亚**：番茄 + 海鲜 + 面食（pizza / spaghetti alle vongole / mozzarella di bufala）

理解这些区别会让吃意大利菜变成地理课。每个城市点本地菜，不要点外地菜（罗马不点 carbonara 之外的面、那不勒斯吃比萨不吃 risotto）—— 这是意大利人的不成文规则。
